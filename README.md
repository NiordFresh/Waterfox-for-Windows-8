# Waterfox for Windows 8

Modified Waterfox 6.5.11 (128) and Waterfox 6.6.7 (140) for Windows 8+ 

6.5.11 (128):
Windows 8.1: Stable.
Windows 8: Quite stable.

6.6.7 (140):
Windows 8.1: Quite stable.
Windows 8: Further observations are needed, not very stable at the moment.

> msvcp140.dll and vsuntime140.dll borrowed from the "Firefox for Windows 7" project by https://github.com/e3kskoy7wqk
